## **Project Overview**

The **Customer Sales Transactions** visualization project is designed to offer insights into various aspects of sales data, including:

* Customer purchasing behaviors
* Product performance across different categories
* Store-level sales analysis

The dashboard incorporates key metrics such as total sales, number of transactions, customer demographics, product categories, and store performances. It uses a variety of interactive visual elements like bar charts, line graphs, heat maps, and filters to allow users to explore the data from multiple perspectives.

---

## **Data Overview**

The dataset for this visualization consists of the following key components:

* **Customer Data**: Information about customers, which includes their IDs and demographics such as age, gender, and location.
* **Transaction Data**: Transaction details, including transaction ID, sales amount, product details, and transaction date.
* **Product Data**: Product IDs, names, and categories.
* **Store Data**: Store types, , and  sales data.

Dataset folder contains the required files needed for visualization.
